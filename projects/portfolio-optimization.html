Portfolio Optimization

Modern Portfolio Theory & Asset Allocation

```

Project Overview

This project explores portfolio construction using Modern Portfolio Theory. The objective was to determine optimal asset allocations that maximise return while minimising risk through diversification.

Problem Being Solved

Investors often struggle to balance risk and reward. This project addresses that challenge by mathematically identifying portfolios that lie on the Efficient Frontier.

Methodology

Expected returns, volatilities, and covariance matrices were used to generate thousands of portfolio combinations. Sharpe Ratios were calculated to identify optimal allocations.

Results

The project produced Efficient Frontier visualisations, minimum variance portfolios, and maximum Sharpe Ratio portfolios. These outputs demonstrate the power of quantitative portfolio construction.
```
