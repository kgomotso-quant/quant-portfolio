Value at Risk Engine

Risk Analytics & Portfolio Risk Management

```

Project Overview

This project was developed to measure portfolio risk using Value at Risk (VaR), one of the most widely used risk metrics in modern finance. The objective was to quantify the maximum expected loss of a portfolio under normal market conditions within a specified confidence level and time horizon.

Problem Being Solved

Financial institutions need a systematic way to understand downside exposure. Without proper risk measurement, portfolios can accumulate hidden vulnerabilities. This project provides a framework to estimate potential losses before they occur.

Methodology

Historical returns were analysed to construct return distributions. Multiple approaches were investigated including Historical VaR, Parametric VaR, and Monte Carlo VaR simulations.

Results

The model successfully quantified downside risk and demonstrated how portfolio losses vary under different market conditions. Results were visualised using loss distributions and confidence intervals.
```
