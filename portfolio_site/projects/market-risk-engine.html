Featured Project

Advanced Market Risk Engine

Python risk analytics project comparing VaR and Expected Shortfall under normal and fat-tailed assumptions, with dashboard reporting and model diagnostics.

What It Does

  • Calculates portfolio returns from market price data.
  • Estimates VaR using Historical, Gaussian, Student-t, and Monte Carlo models.
  • Calculates Expected Shortfall for tail-loss analysis.
  • Backtests VaR breaches with a traffic-light style result.
  • Shows stress losses, drawdowns, portfolio growth, and return distributions.

Tech Stack

  • Python
  • pandas, NumPy, SciPy
  • matplotlib
  • Streamlit
  • pytest

Key Result

The project shows that Gaussian assumptions can materially understate extreme downside risk. Student-t and historical approaches capture fatter tails and produce higher Expected Shortfall estimates at the 99% confidence level.

CV Summary

Built a Python market risk engine and Streamlit dashboard to estimate VaR and Expected Shortfall using Historical, Gaussian, Student-t, and Monte Carlo methods, with backtesting, stress testing, drawdown analysis, and distribution diagnostics.
